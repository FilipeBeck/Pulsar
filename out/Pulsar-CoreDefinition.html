Class: CoreDefinition

Pulsar~CoreDefinition

(abstract) new CoreDefinition(Superclass, Superclasses)

Definição base para todas as classes criadas com o framework. Embora seja documentada como uma classe abstrata com membros estáticos, isto deve ser entendido em um contexto meta-linguístico. As classes criadas pelo framework com Pulsar.class podem ser vistas como uma instância de uma definição núcleo. As propriedades serão aplicadas através de CoreDefinition.call em cada classe definida pelo programador, e na classe Object durante a inicialização do framework.
