Class: CoreDefinition

Pulsar~CoreDefinition

(abstract) new CoreDefinition(Superclasses)

Definição base para todas as classes criadas com o framework. Embora seja documentada como uma classe abstrata com membros de instância, isto deve ser entendido em um contexto meta-linguístico. As classes criadas pelo framework com Pulsar.class podem ser vistas como uma instância de uma definição do núcleo. As propriedades serão aplicadas através de CoreDefinition.call em cada classe definida pelo programador, e na classe Object durante a inicialização do framework (com Superclass = undefined, Superclasses = []).

Methods

(inner) bodyDefiner(rules)

Define as propriedades do corpo
Parameters:
Name Type Description
rules String As regras para a definiçào. Pode ser qualquer combinação permitida entre os seguintes identificadores:
  • public: Membro público (default), acessível através de 'this'. Precisa ser a primeira declaração. Imcompatível com private
  • private: Membro privado, acessível através de '$(this)'. Precisa ser a primeira declaração. Imcompatível com public e static
  • static: Membro de classe. Precisa vir antes de var ou let. Imcompatível com private
  • var: Membro mutável. Imcompatível com let e métodos
  • let: Membro imutável. Imcompatível com var, propriedades computadas e métodos
Source:
  • core.js, line 279
Returns:
Um objeto vazio onde será inserida a propriedade ou método
Example
Pulsar.class('MyClass', Object, function($)
{
  $('public var').a = 10;  // this.a
  $('private let').b = 20; // $(this).b
  $('static var').c = 30;  // MyClass.c
})
