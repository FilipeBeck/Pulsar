import(modulePath, baseType)

Importa o módulo especificado, se já não foi importado. Para essa função trabalhar adequadamente, é necessário que o módulo contenha apenas uma classe, e com o mesmo nome do arquivo (sem a extensão).
Parameters:
Name Type Description
modulePath String A URL do módulo. Se o módulo já foi importado, simplesmente retorna
baseType String Base para a procura do módulo. Pode ser 'relative' (relativo ao módulo atual - default, se não for especificado) ou 'absolute' (relativo ao index).

$.import('MyClass.js') // var myClass = new MyClass()
